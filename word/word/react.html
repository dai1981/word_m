react

動詞
発音：/riˈækt/（リアクト）
中学生〜 英検準2級 CEFR B1 理科
反応する・反発する
📖 主な意味
  • 1 【動詞】反応する・対応する react to bad news（悪い知らせに反応する）。
  • 2 【動詞】（化学）反応する 理科で「物質が化学反応を起こす」意味でも使う。
💡 使い方・例文
① 「（人が）反応する」として
react to〜（〜に反応する）の形でよく使います。
How did she react to the news?
彼女はその知らせにどう反応しましたか。
② 「（物質が）化学反応する」として
理科の文脈で物質どうしの反応を表します。
Sodium reacts violently with water.
ナトリウムは水と激しく反応する。
⚠️ 「〜に反応する」は react to。react with は化学で「〜と反応する」。前置詞で意味が変わる。
⚠️ 名詞は reaction（反応）。形容詞・名詞の reactor は「原子炉」。
🏛 語源・歴史

ラテン語の re-（再び・戻って）と agere（行う・動かす）が合わさった語で、「action（行動）に対して action を返す」＝反応するという意味になりました。
