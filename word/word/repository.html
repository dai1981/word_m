repository

名詞
発音：/rɪˈpɑːzɪtɔːri/（リポジトリー）
高校生〜 英検2級 CEFR B2 もの
保管所・貯蔵庫・宝庫／（IT）リポジトリ
📖 主な意味
  • 1 【名詞】保管所・貯蔵庫・宝庫 ものや情報を集めて保管する場所。a repository of knowledge（知識の宝庫）。
  • 2 【名詞】（IT）リポジトリ プログラムやデータを保存・管理する場所。
💡 使い方・例文
① 「保管所・宝庫」として
情報・物を集めて保管する場所を表します。
The library is a repository of historical documents.
その図書館は歴史的文書の保管所だ。
② IT「リポジトリ」として
コードやデータの保存・管理場所を表します。
The code is stored in a public repository.
そのコードは公開リポジトリに保存されている。
⚠️ repository は『保管所・宝庫』。a repository of＋名詞（〜の宝庫）の形が多い。
⚠️ IT では『リポジトリ（バージョン管理されるコード保管庫）』。略して repo。
🏛 語源・歴史

ラテン語 repositorium（reponere「しまっておく」から＝保管所）に由来します。deposit・position と同じ pos-（置く）の語根で、「（物を）置いておく場所」が原義です。
