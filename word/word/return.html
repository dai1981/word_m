return

動詞 / 名詞
発音：/rɪˈtɜːrn/（リターン）
小学生〜 英検4級 CEFR A2 基本動詞
戻る・帰る・返す／帰り・返却
📖 主な意味
  • 1 【動詞】戻る・帰る 自動詞。return home（帰宅する）。return to（〜に戻る）。
  • 2 【動詞】返す・返却する 他動詞。return the book（本を返す）。
  • 3 【名詞】帰り・返却・見返り in return（お返しに）。
💡 使い方・例文
① 「戻る・帰る」として
return to＋場所で「〜に戻る」を表します。
She returned to Japan after two years.
彼女は2年後に日本に戻った。
② 「返す」として
借りたものを返す他動詞用法です。
Please return the book by Friday.
金曜日までに本を返してください。
⚠️ 「〜に帰る」は return to＋場所。ただし return home は to をつけない（home は副詞）。
⚠️ return back は冗長。return だけで『戻る』の意味を含むので back は不要。
🏛 語源・歴史

古フランス語 retourner（re-「再び」＋ tourner「回る・向ける」＝向きを変えて戻る）に由来します。turn（回る）と同じ語根で、「向きを変えて元へ」が原義です。
